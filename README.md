ProjectStarter
===

Get straight to coding by automagically creating a project folder with little to no effort.

# How to install

```
pip3 install virtualenv
python3 -m venv venv
source ./venv/bin/activate
pip3 install -r requirements.txt
```

# How to use

```
python3 ./project.py --help
```
